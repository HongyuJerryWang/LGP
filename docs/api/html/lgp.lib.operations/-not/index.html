Not

class Not : UnaryOperation<Double>

Performs the bitwise not function on a single Double argument.

To achieve this, the double input value is converted to a boolean by mapping zero values to false and non-zero values to true.

The resulting boolean is then mapped back to a double value such that true maps to 1.0 and false maps to 0.0.
