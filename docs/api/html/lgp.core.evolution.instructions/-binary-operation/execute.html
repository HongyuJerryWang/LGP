execute

open fun execute(arguments: Arguments<T>): T

Applies the operations function to the argument(s) given.

If the number of arguments given does not match the operations arity, then an exception will be thrown.

Parameters

arguments - A collection of arguments (should be at most 2).

Exceptions

ArityException - When the number of arguments does not match the arity.

Returns
A mapping of the arguments to some value.
