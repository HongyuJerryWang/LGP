Arguments(arguments: List<Argument<T>>)

A simple collection of Arguments.

The sole purpose of this wrapper class is to make the syntax of functions for operations a bit cleaner. For example, we can write:

// A simple cosine function definition
val cos: (Arguments<Double>) -> Double = { args -> Math.cos(args.get(0)) }

Instead of:

// A verbose cosine function definition
val cos: (List<Argument<Double>>) -> Double = { args -> Math.cos(args.get(0)) }

Parameters

T - The type of the arguments.
