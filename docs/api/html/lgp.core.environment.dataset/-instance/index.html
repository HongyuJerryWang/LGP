open class Instance<out TAttribute>

An instance in a Dataset.

Instances are made up of a collection of Attributes. A particular attribute is marked as the class attribute and the rest are marked as input features. The indices of these features can be controlled via the Dataset the instance belongs to.

NOTE: Similarly as for Datasets, custom implementations of samples can be defined but some functionality is marked so that it is not able to be overridden (see any functions marked final).
