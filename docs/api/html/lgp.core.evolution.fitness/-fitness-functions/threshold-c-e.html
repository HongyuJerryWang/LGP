thresholdCE

@JvmStatic val thresholdCE: (threshold: Double) -> FitnessFunction<Double>

Threshold Classification Error fitness function.

This fitness function works by counting the number of outputs which have an error difference greater than a given threshold. Error differences which fall beneath the threshold are considered to be "good" solutions.

The field expects to be passed a threshold for differentiating good/bad solutions, specified through the threshold parameter.
