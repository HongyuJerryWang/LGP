BaseInstruction

class BaseInstruction<T> : Instruction<T>

A built-in offering of the Instruction type.

This instruction is simple in its representation and execution: - It consists of a single operation that operates on a set of operand registers and stores the result in a single destination register. - It provides the ability to be exported as a C-style instruction (e.g. "r1 = r1 + r2")
