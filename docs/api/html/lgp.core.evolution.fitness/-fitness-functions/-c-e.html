@JvmStatic fun CE(mappingFunction: (Double) -> Double): FitnessFunction<Double>

Classification Error fitness function for programs that operate on doubles.

Evaluates a program on a set of fitness cases using classification error on the actual and expected class values. The actual value is whatever is in register 0 after program execution mapped using the mapping function given.

Parameters

mappingFunction - A function that maps the continuous program output to a discrete class value.
