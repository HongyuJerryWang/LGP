Argument(value: T)

An Argument is essentially just the value of a particular Register, but in the context of an lgp.core.evolution.instructions.Operation.

When the value of a register is to be passed as an operand to some operation (and the function that operation applies), it should be mapped to an argument type.

Parameters

T - Type of value this argument represents.
