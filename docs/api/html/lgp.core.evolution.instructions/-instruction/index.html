Instruction

abstract class Instruction<T> : Module

An instruction in an LGP program.

Instructions can be implemented in any way but they must operate on a set of registers, with one register as output and a set of registers as operands.

The registers the instruction operates on are specified by an index.

T - The type of data the instruction operates on.
