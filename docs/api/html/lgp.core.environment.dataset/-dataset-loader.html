interface DatasetLoader<TData> : ComponentLoader<Dataset<TData>>

An extended ComponentLoader that loads a Dataset.

The source of the data set is controllable by any implementor, all that is required is that the raw data is gathered into a Dataset instance, while satisfying the type constraint TData for individual attributes in the data set.

Parameters

TData - Type of attributes in the data set.
