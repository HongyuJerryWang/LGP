thresholdCE

@JvmStatic fun thresholdCE(threshold: Double): FitnessFunction<Double>

Threshold Classification Error fitness function.

This fitness function works by counting the number of outputs which have an error difference greater than threshold. Error differences which fall beneath threshold are considered to be "good" solutions.

Parameters

threshold - The threshold for differentiating good/bad solutions.
