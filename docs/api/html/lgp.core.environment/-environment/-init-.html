Builds an environment with the specified construction components.

Parameters

configLoader - A component that can load configuration information.

constantLoader - A component that can load constants.

datasetLoader - A component that can load the dataset for an LGP system.

operationLoader - A component that can load operations for the LGP system.

defaultValueProvider - A component that provides default values for the registers in the register set.

fitnessFunction - A function used to evaluate the fitness of LGP programs.

initialise - Whether or not initialised components should be initialised automatically.
