Guide
*****

This document describes the design of the LGP system and how to use it to get started with LGP.

It is assumed that the reader has prior familiarity with LGP, or has read the section on :ref:`lgp-ref`.

For a complete, low-level API reference, see the `dokka API documentation <https://jeds6391.github.io/LGP/>`_, as this document will primarily discuss the concepts of the API and how to use them at a high level.
